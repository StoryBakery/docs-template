### GetSize {#GetSize}

<div class="sb-ref-heading-actions"><a class="sb-ref-source sb-ref-source-inline" href="https://github.com/StoryBakery/bakerywave/blob/main/src/DataObjectTypeInfos.luau#L79"><svg viewBox="0 0 24 24" aria-hidden="true" focusable="false" class="sb-ref-source-icon"><path fill="currentColor" d="M8.7 16.6 4.1 12l4.6-4.6L7.3 6l-6 6 6 6 1.4-1.4zm6.6 0L19.9 12l-4.6-4.6L16.7 6l6 6-6 6-1.4-1.4zM10 19l4-14h2l-4 14z"/></svg></a></div>

```lua
(value: any) -> number
```

`Size`가 `"Undefined"`일 때, 실제 직렬화 크기를 결정하는 함수입니다.
예: `string` 은 문자열 길이에 따라 달라지므로 `#value + 2` 같은 식으로 계산할 수 있습니다.

`Size` 가 고정되어있다면, 해당 `Size` 를 반환합니다.
